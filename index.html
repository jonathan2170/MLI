API 模式需要 Anthropic API Key。請到「設定」分頁輸入,或切換為「手動模式」(免 API Key,使用 Claude Pro)。

📋 手動模式 (使用 Claude Pro)

三步驟完成,完全免費(用你既有的 Claude Pro 月費)。

1
複製提示詞
點下方按鈕複製,提示詞會自動帶上完整對應規則。
2
到 Claude.ai 解析
開新對話 → 把保障彙整表 PDF 拖進去 → 貼上剛剛複製的提示詞 → 送出。
完成後把 Claude 回覆的整段 JSON 複製起來。
📱 開啟 App 🌐 網頁版
3
貼回 JSON 結果
把 Claude 回應的 JSON 整段貼到下方,點解析即可。

尚無解析結果
請先到「上傳」分頁解析一份 PDF
